# Security

If you believe you have found a security vulnerability in Terramate, please report it responsibly.

**Do not open a public GitHub issue** for security-sensitive bugs.

Instead, please report it here:

- **[Report a security vulnerability](https://github.com/terramate-io/terramate/security/advisories/new)** — use GitHub’s Security Advisories to submit a private report.

We will respond as soon as we can and will keep you updated on the status of the report.
